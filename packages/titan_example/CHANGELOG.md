# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-02

### Added
- **Questboard** example app — story-aligned showcase of all Titan features
- Hero management with `HeroPillar` (Core, Derived, Strike patterns)
- Quest system with `QuestPillar` (state machines, Herald events)
- Guild management with `GuildPillar` (collections, computed aggregates)
- Atlas routing with Passages, Sanctums, and Sentinels
- Beacon-based dependency injection across all screens
- Vestige widgets demonstrating reactive UI rebuilds
- Confluence multi-consumer widget examples
- Epoch undo/redo on hero stat editing
- Scroll form validation on hero creation
- Chronicle structured logging throughout
- Vigil error tracking integration
- Full navigation with bottom tabs and nested routes
